Simple project using SFML for window + event management and OpenGL to render and shade a triangle.
==================================================================================================

# Libraries:
`OpenGL` (libgl1-mesa-dev ...) `GLEW` (libglew-dev) `SFML` (libsfml-dev)

# Toolchain
`g++ 9.3.0`

# Build
`make`
